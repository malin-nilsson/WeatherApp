# Weather App

A simple weather app built with Vue.js that fetches weather from Open Weather Map.

![](assets/screenshot.jpg)

- 
## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
